CSV Data Analyzer
Ferramenta para análise de dados em arquivos CSV, desenvolvida para facilitar a manipulação, visualização e extração de informações relevantes de planilhas e bancos de dados em formato CSV.

Funcionalidades
Carregamento de arquivos CSV de forma simples e rápida
Visualização dos dados em tabelas
Filtragem e ordenação de colunas
Geração de estatísticas descritivas (média, mediana, moda, etc.)
Exportação de resultados para novos arquivos CSV
Interface intuitiva para interação com os dados
Como usar
Clone o repositório:

bash
git clone https://github.com/alex-des-santos/CSV-Data-Analyzer.git
Instale as dependências (ajuste conforme a tecnologia utilizada):

bash
# Exemplo para Python
pip install -r requirements.txt
Execute o analisador:

bash
# Exemplo para Python
python main.py
